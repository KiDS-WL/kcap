To use the SPTpol likelihood, download this file:
https://pole.uchicago.edu/public/data/henning17/sptpol_cosmomc_nov16_v1p1.tar.gz

extract it with 
tar -zxvf sptpol_cosmomc_nov16_v1p1.tar.gz
and move the "data" subdirectory into this directory (next to this readme file).
